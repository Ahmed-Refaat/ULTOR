# App Icon

Your app icon lives here as **icon.png**.

- **Size:** 512×512 or 1024×1024 pixels (recommended)
- electron-builder converts it to .icns (macOS) and .ico (Windows) automatically

**To use your own logo:**
1. Replace `build/icon.png` with your PNG (or drop your file here and rename to `icon.png`)
2. Run `npm run build`
3. The DMG and .app will use your icon

Currently using `public/ultor-logo.png` as the default. Replace `icon.png` with your own file when ready.
